# 倪海厦知识图谱

一个面向学习者的倪海厦公开教学资料结构化知识图谱。

在线访问：https://zenoyang-ai.github.io/nihaixia-knowledge-graph/

## 项目简介

本项目把倪海厦相关公开教学资料整理为：
- 70 个知识节点
- 330 条关系
- 五层知识架构
- 人纪 / 天纪学习路径
- 天纪提示词工具包

## 功能

- 交互式知识图谱（D3 力导向图，支持拖拽缩放）
- 节点知识卡片（定义、核心理解、学习路径、安全边界）
- 人纪 / 天纪学习路径
- 全局搜索
- 天纪提示词工具包（八字、紫微、易经、小六壬）

## 内容边界

本项目仅供学习研究，不构成医疗建议。
不提供诊断、处方、剂量、服法等可执行用药信息。
不公开付费课程、医案全文、个人隐私资料和高风险原始材料。
