# 内容声明

## 项目说明

本仓库是倪海厦知识图谱的开源版本，包含知识图谱节点、学习路径、主题文章，以及部分转换后的公开资料。

## 内容分级

### A 类：可公开

- 古籍原文：黄帝内经、伤寒论、金匮要略、神农本草经等古典医书原文
- 作者原创：知识图谱节点、学习路径、主题文章、资料清单、来源索引
- 转换说明：工具脚本、转换日志、质量标记

### B 类：谨慎公开

- 古籍的现代整理版、注解版、白话版
- 网络公开流传的课程文字稿
- GitHub 9527qingfeng 等第三方整理资料

处理方式：保留 attribution，标记来源，只放清洗后的 Markdown。如权利人要求，可删除。

### C 类：默认不公开全文

- 医案原文
- 临床案例
- 方剂详细讲解
- 含患者经历、年龄、时间线、具体治疗过程的文本

处理方式：只公开节点摘要、学习框架、脱敏片段。全文不在本仓库公开。

### D 类：不公开

- 百度网盘分享链接、提取码、token
- 个人笔记、任务日志、系统协议
- 付费课程录音/字幕的完整整理稿
- 含隐私的地址、电话、患者信息
